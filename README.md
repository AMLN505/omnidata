# Omnidata
## О проекте
Сайт для компании-производителя ПО по автоматизации внутренний процессов ритейлеров.

## Моя роль в проекте
Frontend-разработчик

## Инструкция по запуску
1. Скачать сборку из ветки main
2. Открыть проект в IDE
3. Установить пакеты с помощью команды в терминале "npm i"
### DEV-версия
- Ввести в терминал команду "gulp dev"
### BUILD-версия
- Ввести в терминал команду "gulp build"
